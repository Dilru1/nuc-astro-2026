# Nuc-Astro-2026
Curriculum materials and computational labs for the Nuclear Astrophysics Module at the University of Colombo.

## Course Overview
This repository contains the source material for the course, including:
- **Textbook Modules:** Theoretical chapters covering nuclear mass, binding energy, and nucleosynthesis.
- **Computational Labs:** Jupyter-based simulations for modeling nuclear stability and reaction rates.
- **Interactive Tools:** Google Colab links for real-time parameter tuning.
