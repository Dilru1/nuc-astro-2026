# න්‍යෂ්ටික තාරකා භෞතික විද්‍යාව 2026 වෙත ඔබව සාදරයෙන් පිළිගනිමු
[English Version - ඉංග්‍රීසි අනුවාදය](../index.html)

මෙය න්‍යෂ්ටික තාරකා භෞතික විද්‍යාව සඳහා වන නිල පාඨමාලා වෙබ් අඩවියයි. මෙහිදී ඔබට දේශන සටහන්, පැවරුම් සහ කියවීම් ද්‍රව්‍ය ලබා ගත හැකිය.

## පාඨමාලා දළ විශ්ලේෂණය
- **දේශක:** දෙහිවලගේ දොන් දිල්රුවන්
- **අධ්‍යයන වාරය:** 2026 වසන්තය
- **අරමුණ:** විශ්වයේ මූලද්‍රව්‍ය නිර්මාණය වීම පිළිබඳව ගවේෂණය කිරීම.
